¿Qué es la Programación Estructurada?

La Programación Estructurada es un paradigma de programación que facilita la escritura y mantenimiento del código mediante el uso de estructuras lógicas bien definidas.

Asi mismo la programacion estructurada es una tecnica de programación que organiza el código en estructuras logicas.

Esto facilita la comprension y el mantenimiendo del codigo.

1. Variables y Tipos de Datos

Las variables son espacios en memoria para almacenar datos. Cada variable tiene un tipo de dato asociado.

                
// Ejemplo en C
int edad = 25;
float altura = 1.75;
char inicial = 'A';

Variables Numericas

Almacenan numeros o decimales

Varianbles de Texto

Almacenan cadenas de caracteres

Variables booleanas

Almacenan valores de true o false

Variables de Caracter

Representan simbolos o caracteres

2. Estructuras de Control

Las estructuras de control permiten modificar el flujo del programa según condiciones.

                
// Ejemplo de if-else en C
if (edad >= 18) {
    printf("Eres mayor de edad.");
} else {
    printf("Eres menor de edad.");
}

3. Funciones y Modularidad

Las funciones permiten dividir el código en bloques reutilizables, facilitando la lectura y mantenimiento.

                
// Ejemplo de función en C
void saludar() {
    printf("Hola, bienvenido a la programación estructurada!\n");
}
